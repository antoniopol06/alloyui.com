How Dropdown button works

Dropdown button uses focus manager widget (The use of widgets are mentioned in create buttons guide) that allows the dropdownÂ´s behavior

The behavior is set by these props component: 

	
	getDefaultProps: function() {
        return {
            circular: false,
            descendants: '.ae-toolbar-element',
            keys: {
                dismiss: [27],
                dismissNext: [39],
                dismissPrev: [37],
                next: [40],
                prev: [38]
            }
        };
	}

Circular: Boolean. True value allow you to pass through the options with the related keycode (next and prev), when you reach the last item using next key, the next option using the same key is the first option and viceversa. False value means that when you reach the last item using next key, you need to use prev key to navigate through the previous options.

Descendants: String. The CSS selector used to define the elements that should be handled by keycodes.

Keys: Object. This object contains keycodes to handle the navigation through the options.

Dismiss key cancels the dropdown and the toolbar return to the previous state.

DismissNext | DismissPrev keys cancel the dropdown but the navigation continues to the next|previous button of the toolbar.

Next | Prev keys allow you to navigate between dropdown options.
