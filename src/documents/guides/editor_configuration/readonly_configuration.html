Passing readOnly configuration option to the editor

The readOnly configuration is configured via a common readOnly property, like this: 

	var editor = AlloyEditor.editable('editable', {
    	readOnly: [true|false]
	});

The readOnly is the property which contains a boolean value. true means that editor is disabled and link element (a tag) works as default behavior, and, false means that editor is enabled to modify the text.

This property is available to be changed at anytime using the following code: 


	editor.setReadOnly([true|false]);
